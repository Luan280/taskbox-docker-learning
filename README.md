## Objetivo

- Aprender os principais conceitos de Docker e Docker Compose
- Praticar Git, GitHub, testes e integração contínua
- Aprender sem copiar códigos prontos e sem depender de vibe coding
- Ler e consultar as documentações oficiais
- Desenvolver o pensamento lógico e escrever meus próprios códigos
- Entender e aplicar boas práticas de desenvolvimento

## O que já aprendi

- Diferença entre imagem e container
- Container como instância de uma imagem
- Ambiente virtual Python
- Dependências diretas e transitivas
- Fluxo básico de Git
- Aplicação Flask e rotas HTTP
- Configuração do ambiente de desenvolvimento

## Tecnologias utilizadas

- Python
- Flask
- Git
- GitHub

## Como executar atualmente

1. Criar o ambiente virtual:

```bash
    python3 -m venv .venv
```

2. Ativar o ambiente:

```bash
    source .venv/bin/activate
```

3. Instalar as dependências:

```bash
    python -m pip install -r requirements.txt
```

4. Executar o Flask:

```bash
    flask --app app run --debug
```
